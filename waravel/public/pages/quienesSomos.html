Quiénes Somos

Somos un grupo de estudiantes del segundo curso del ciclo formativo de Desarrollo de Aplicaciones Web (DAW). Este proyecto es el resultado de nuestro trabajo en equipo, titulado WARAVEL, una plataforma creada para facilitar la compra y venta de productos de segunda mano.

WARAVEL está desarrollado utilizando Laravel, un potente framework de PHP. Nuestro objetivo es ofrecer a los usuarios una plataforma robusta, segura y fácil de usar para realizar transacciones de productos de segunda mano.

Enfoque Tecnológico

  • Laravel para el desarrollo del backend.
  • PHP como lenguaje de programación principal.
  • Blade para la creación de vistas dinámicas.
  • PostgreSQL para la gestión de bases de datos.
  • Redis para optimización de rendimiento.
  • Docker para la contenedorización y despliegue.
  • Cypress para pruebas de calidad.
